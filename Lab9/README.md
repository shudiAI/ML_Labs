# ARTI308_Lab9
## Decision Trees and Random Forest Project

This project uses loan data to predict whether a borrower will **not fully pay back the loan**.  
The target column is:

`not.fully.paid`

### The project compares two machine learning models:

#### Decision Tree
A Decision Tree is simple and easy to understand.  
However, it can overfit the training data because it depends on one tree only.

#### Random Forest
Random Forest uses many Decision Trees together.  
This usually makes the prediction more stable and more accurate.

### Which Model is Better?
The **Random Forest model is better** than the Decision Tree model.

Random Forest usually performs better because it combines many trees, which reduces overfitting and gives more reliable results.  
Decision Tree is easier to understand, but it may not generalize well on new data.
