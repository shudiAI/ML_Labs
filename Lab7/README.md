# ARTI308_Lab7
In this project, we worked on a fake advertising dataset to predict whether a user will click on an advertisement.

We performed the following steps:

 1-Imported and explored the dataset using pandas (info() and describe()).<br> 
 2-Conducted exploratory data analysis using seaborn (histograms, jointplots, and pairplots) to understand relationships between features. <br>
 3-Selected key numerical features such as: <br>
   Daily Time Spent on Site  <br>
   Age <br>
   Area Income <br>
   Daily Internet Usage <br>
   Male<br> 
 4-Split the data into training and testing sets using train_test_split.<br> 
 5-Built a Logistic Regression model using scikit-learn. <br>
 6-Trained the model and made predictions on the test data. <br>
 7-Evaluated the model using a classification report, achieving around 90% accuracy. <br>
 8-Improved convergence by increasing max_iter when needed.<br>

Overall, the model performed well in predicting whether a user would click on an ad based on their behavior and demographic features.
