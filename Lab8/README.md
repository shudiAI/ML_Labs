# ARTI308_Lab8
### KNN Classification Project

This project applies the K-Nearest Neighbors (KNN) algorithm to classify an anonymous dataset.

#### Steps:
Performed exploratory data analysis to understand feature relationships.<br>
Standardized the data using StandardScaler.<br>
Trained a baseline KNN model.<br>
Evaluated the model using confusion matrix and classification report.<br>
Used the elbow method to find the optimal K value.<br>
Retrained the model using the best K.<br>

#### Result:
The final model achieved better accuracy after tuning the K value.
