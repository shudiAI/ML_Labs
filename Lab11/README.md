### K-Means Clustering Project

This project applies the K-Means clustering algorithm to segment credit card customers based on their financial behavior.

#### Steps:

Performed exploratory data analysis to understand customer spending patterns and feature relationships.<br>
Handled missing values and removed unnecessary columns such as customer ID.<br>
Standardized the dataset using StandardScaler.<br>
Applied the K-Means clustering algorithm to group customers into clusters.<br>
Used the Elbow Method and selected K = 4 as the optimal number of clusters.<br>
Visualized the clusters using PCA for dimensionality reduction.<br>
Analyzed cluster characteristics to understand customer segments.<br>

#### Result:

The final clustering model successfully grouped customers into 4 meaningful customer segments based on their credit card usage behavior, helping provide insights for customer analysis and marketing strategies.
