# ARTI308_Lab4

This lab focuses on data quality assessment and preprocessing using the BMW Car Dataset as part of the ARTI308 Machine Learning course.
The work includes the following tasks:

1-Data Quality Assessment : Checking data types, structure, and consistency issues in the dataset.

2-Missing Value Handling : Detecting missing values and applying median imputation.

3-Outlier Detection & Treatment : Using the Interquartile Range (IQR) method to detect outliers and applying capping to reduce their impact.

4-Data Normalization : Applying Min-Max scaling and Z-score standardization to ensure features are on comparable scales.

5-Correlation Analysis : Generating a correlation heatmap to analyze relationships between numerical features.

6-Dimensionality Reduction (PCA) : Applying Principal Component Analysis to reduce feature dimensions while preserving variance.

All steps are implemented using Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn to ensure the dataset is clean, consistent, and ready for machine learning modeling.
